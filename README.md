# D4Mods Releases

Public releases for Degrees of Lewdity mods.

## Available Mods

### Combat UI Rework
Improves the game's combat interface with:
- Split-view layout with configurable ratio (30-70%)
- Compact mode for mobile devices
- Persistent user settings

## Installation

1. Download the latest `.zip` from [Releases](https://github.com/LeafGithub/D4Mods-Releases/releases)
2. Load via ModLoader (drag & drop or use the mod manager)

## Compatibility

Requires [SugarCube 2 ModLoader](https://github.com/Lyoko-Jeremie/sugarcube-2-ModLoader)
